Type Structure Reference Glossary

Mathematical borrows from principles of OOCSS and promotes minimal styling to stay true to the medium.

Type

The default type scale uses four font sizes, all based on simple math. Smaller screens use less type size contrast. Bigger screens get more dramatic contrast.

If you think restricting yourself to four font sizes is difficult or not ideal for certain interfaces, it may just be that this feels a little unfamiliar. But a type scale like this can help achieve clarity, brevity, and simplicity. After all, it's all about information density, and scrolling is just a form of responsive disclosure.

Use this smaller size for helper text, form labels and meta content. Do you really need to go any smaller?

Default Type Scale:

Algebraic (h1)

Geometric (h2)

Trigonometric (p)

Polynomial (small)

Spacing Utilities

Mathematical uses a suite of spacing utility classes to adjust margin and padding on elements. These utilities are helpful in creating gestalt and prevent unnecessarily editing CSS, which should help speed you up.

Positioning Utilities

The suite of positioning utilities are for when you want to change the document flow, make things overlap, lock chrome, etc.
