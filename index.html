Type

The default type scale uses four font sizes, all based on simple math. Smaller screens use less type size contrast. Bigger screens get more dramatic contrast.

Algebraic (h1)

Geometric (h2)

Trigonometric (p)

Polynomial (small)

Structure

HTML is inherently structured. Use CSS to enhance the experience.

Spacing Utilities

Mathematical uses a suite of spacing utility classes to adjust margin and padding on elements. These utilities are helpful in creating gestalt and prevent unnecessarily editing CSS, which should help speed you up.

Positioning Utilities

The suite of positioning utilities are for when you want to change the document flow, make things overlap, lock chrome, etc.
